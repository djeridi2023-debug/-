# نظام تسيير مكتب التوثيق - الجزائر (AI Notary DZ)

هذا التطبيق مصمم خصيصاً لمكاتب التوثيق في الجزائر، ويدمج تقنيات الذكاء الاصطناعي لتسهيل العمل اليومي والامتثال للقوانين الجزائرية.

## المميزات الرئيسية:
- **إدارة العملاء:** إضافة وتسيير بيانات العملاء (أشخاص طبيعيين ومعنويين) مع دعم رقم التعريف الجبائي (NIF).
- **إدارة العقود:** تسجيل ومتابعة العقود الموثقة (بيع، هبة، تأسيس شركات، إلخ).
- **مساعد الذكاء الاصطناعي:** صياغة العقود تلقائياً باللغة العربية الفصحى القانونية.
- **الامتثال القانوني (AML/KYC):** تنبيهات آلية عند تجاوز المبالغ المالية للحدود القانونية (50 مليون سنتيم) وفق القانون الجزائري.
- **دعم اللغة العربية:** واجهة مستخدم كاملة باللغة العربية مع اتجاه من اليمين إلى اليسار (RTL).

## متطلبات التشغيل:
- نظام تشغيل ويندوز (Windows 10/11).
- تثبيت لغة البرمجة Python (الإصدار 3.10 أو أحدث).
- **هام جداً:** عند تثبيت بايثون، تأكد من تفعيل خيار **"Add Python to PATH"** في نافذة التثبيت الأولى.

## كيفية التثبيت والتشغيل:
1. قم بفك الضغط عن الملف.
2. انقر نقراً مزدوجاً على ملف `setup.bat`. سيقوم هذا الملف تلقائياً بإنشاء بيئة افتراضية (Virtual Environment) وتثبيت جميع المكتبات اللازمة وبناء التطبيق.
3. بعد الانتهاء، ستجد التطبيق في مجلد `dist` باسم `NotaryDZ_AI.exe`.

## إعداد الذكاء الاصطناعي:
للحصول على أفضل النتائج في صياغة العقود، يرجى إنشاء ملف باسم `.env` في المجلد الرئيسي وإضافة مفتاح OpenAI الخاص بك:
```env
OPENAI_API_KEY=your_api_key_here
```

## ملاحظة قانونية:
هذا التطبيق هو نموذج أولي (Prototype) مصمم بناءً على البحث القانوني والوظيفي المقدم. يجب مراجعة العقود المولدة من قبل الموثق المعتمد قبل التوقيع.
